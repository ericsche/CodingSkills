# CodingSkills

A curated collection of **custom skills** built from hands-on development
experience in the **Microsoft ecosystem** — Azure, .NET, Microsoft 365,
Microsoft Graph, and related tooling.

## About

This repository gathers reusable skills, patterns, and automation distilled
from real-world projects. The goal is to capture practical, battle-tested
know-how so it can be reused across future work instead of being rediscovered
each time.

## Focus areas

- **Azure** — App Service, Functions, Static Web Apps, Container Registry, and
  deployment automation.
- **.NET** — modern .NET SDK tooling, isolated Azure Functions, and Blazor.
- **Microsoft 365 & Copilot** — declarative agents and M365 integrations.
- **Microsoft Graph** — authentication and API automation.
- **Identity & Auth** — Entra ID (Azure AD), Easy Auth, and app registrations.
